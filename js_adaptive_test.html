Adaptive JavaScript Skills Test

A computer-adaptive quiz of core programming concepts in JavaScript. Each question gets harder when you answer correctly and easier when you slip — so the test homes in on your true skill level.

How it works

This is a computer-adaptive test (CAT). You'll answer a fixed set of multiple-choice questions about basic computer science and JavaScript. The difficulty of each next question depends on how you did on the previous one.

  • Topics: variables & types, operators, conditionals, loops, functions, scope, arrays, objects, and a few classic gotchas.
  • 5 difficulty levels, from Novice to Expert. You start in the middle.
  • Answer right → next question gets harder. Answer wrong → it gets easier.
  • You get 45 seconds per question — if the timer runs out it counts as a miss, so keep moving.
  • Your final score is an ability estimate (0–100), not just a raw percentage, so it reflects the difficulty of what you actually got right.
  • Take it as many times as you like — your attempts are saved locally and compared to check test–retest reliability (validity).
  • Works in any modern browser, desktop or mobile. Nothing is uploaded; results stay on your device.
